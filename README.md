# Executive Sales Dashboard (Excel VBA)

A professional Excel dashboard for sales performance reporting built using Excel, Power Query, Pivot Tables and VBA.

## Features

- Executive KPI Dashboard
- Sales Trend
- Top Products
- Top Customers
- Regional Sales Analysis
- Interactive Slicers
- Refresh Button (VBA)
- Export to PDF

## Tools

- Microsoft Excel
- VBA
- Power Query
- Pivot Table
